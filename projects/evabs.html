Education game for learning english in playful way with 2D game.

March 2024

July 2024

This project was my second academic requirement for graduation, serving as my final project or thesis. It involved the development of a 2D Android-based game designed to help elementary school students improve their English vocabulary through gameplay. The mobile game application was specifically developed for students at SDN Ciwalen in Ciwalen, Indonesia. The game was built using C# as the programming language and Unity as the development platform. I successfully completed the project within the planned timeline and received positive feedback from users. Based on the report I compiled after conducting field implementation, I managed to improve the students' English scores by 13.2% using a pre-test and post-test method.

The game features two modes: Learning and Playing. The Learning mode provides users with study materials, while the Playing mode serves as the main gameplay experience. The game consists of 7 levels, each with different gameplay mechanics. These include quizzes, drag-and-drop, word scrambles, Wordle-style puzzles, and hidden object challenges.
